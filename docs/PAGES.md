# PAGES.md — 画面構成

> ⚠️ **AIへの指示:** 画面を変更・追加する前に必ずこのファイルを確認すること。
> このファイルにない画面は作成しないこと。変更後はこのファイルも更新すること。

## ナビゲーション構成

```
ヘッダー（Header.astro）
├── ロゴ → /
├── ホーム → /
├── 記事一覧 → /blog
├── About → /about
└── SNSリンク（X, GitHub）
```

## ページ一覧

| パス | ファイル | 役割 | ナビ表示 |
|------|----------|------|---------|
| `/` | `pages/index.astro` | トップページ | ホーム |
| `/blog` | `pages/blog/index.astro` | 記事一覧 | 記事一覧 |
| `/blog/[slug]` | `pages/blog/[...slug].astro` | 記事詳細 | - |
| `/about` | `pages/about.astro` | 自己紹介 | About |

## コンポーネント構成

| コンポーネント | 役割 | 使用箇所 |
|---------------|------|---------|
| `Header.astro` | グローバルナビゲーション | 全ページ |
| `HeaderLink.astro` | ナビリンク | Header内 |
| `Footer.astro` | フッター | 全ページ |
| `BaseHead.astro` | meta/OGP | 全ページ |

## 変更ルール

1. **新しいページを追加する場合**
   - このファイルに追記してから実装
   - ナビに追加が必要か確認

2. **ナビゲーションを変更する場合**
   - `Header.astro` のみ編集
   - このファイルを更新

3. **既存ページを削除する場合**
   - 先にこのファイルから削除
   - ナビからリンクを削除
   - ファイルを削除
