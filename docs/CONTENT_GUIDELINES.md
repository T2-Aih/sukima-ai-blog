# コンテンツガイドライン

## コアコンセプト

**「5分で読んで10分で実践」**

すべての記事は：
- **5分以内**で読める
- **10分以内**にスマホで実践できる

---

## 記事の必須要素

### 1. 読了時間の明示
- 記事冒頭に「読了: 5分」を表示
- 1500〜2000字を目安

### 2. 実践ステップ
- 必ず「今すぐできること」セクションを含む
- スマホで完結する手順
- スクリーンショット付きで迷わない

### 3. 構成テンプレート

```markdown
# タイトル

読了: 5分 | 実践: 10分

## この記事でできること
- 箇条書きで3点以内

## 背景（任意）
- なぜこれが必要か（100字以内）

## 実践ステップ
1. ステップ1（スクショ付き）
2. ステップ2
3. ステップ3

## まとめ
- 今日やったこと
- 次に試すこと（次回記事への導線）
```

---

## 禁止事項

| ❌ やらない | 理由 |
|-------------|------|
| 長い前置き | 読者の時間を奪う |
| PC必須の手順 | ターゲットは通勤中のビジネスマン |
| 理論だけの記事 | 実践がないと価値がない |
| 10分で終わらない手順 | コンセプト違反 |

---

## 品質チェックリスト

記事公開前に確認：

- [ ] 5分で読めるか（1500〜2000字）
- [ ] 10分で実践できるか
- [ ] スマホだけで完結するか
- [ ] 「今すぐできること」があるか
- [ ] 次のアクションが明確か
